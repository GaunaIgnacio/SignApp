ARCHIVOS PREEXISTENTES PARA LA VERSIÓN PÚBLICA

Esta carpeta contiene los archivos preentrenados que se incluirán en la APK pública:

1. gesture_model.ml: Modelo de reconocimiento de señas preentrenado
2. gesture_data/: Carpeta con los archivos JSON de las señas grabadas

Estos archivos se copiarán automáticamente a los archivos internos de la aplicación
cuando el usuario instale y ejecute la aplicación por primera vez.

Nota: Para preparar la versión pública, copie el archivo gesture_model.ml y la carpeta
gesture_data del archivo ZIP exportado a esta ubicación antes de compilar la APK.
